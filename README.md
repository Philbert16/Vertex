# Vertex - Job Portal Website

A modern job portal website connecting employers with job seekers. Built with HTML, CSS, and JavaScript.

## Project Structure

```
Vertex/
├── index.html          # Main landing page
├── employers.html      # For employers seeking talent
├── findjobs.html      # Job search page
├── getintouch.html    # Contact page
├── hire.html          # Hiring services
├── jobseekers.html    # For job seekers
├── morejobs.html      # Additional job listings
├── register.html      # User registration
├── schedule.html      # Scheduling/appointments
├── sign-in.html       # User login
└── images/            # Image assets
```

## Pages Overview

| Page              | Description                                              |
| ----------------- | -------------------------------------------------------- |
| `index.html`      | Main landing page with hero section and featured content |
| `employers.html`  | Employer-focused landing for posting jobs                |
| `findjobs.html`   | Job search and filtering interface                       |
| `jobseekers.html` | Resources and tools for job seekers                      |
| `hire.html`       | Premium hiring services and features                     |
| `getintouch.html` | Contact form and company info                            |
| `register.html`   | New user account creation                                |
| `sign-in.html`    | Existing user authentication                             |
| `schedule.html`   | Appointment scheduling                                   |
| `morejobs.html`   | Extended job listings                                    |

## Features

- Modern dark theme UI with red accent colors
- Responsive design for all screen sizes
- Smooth animations and transitions
- Glassmorphism effects
- Interactive navigation
- Form validation
- Job filtering and search capabilities

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, animations, flexbox/grid
- **JavaScript** - Interactive functionality
- **Google Fonts** - Typography (Arial, sans-serif fallback)

## Getting Started

1. Clone or download the repository
2. Open `index.html` in a web browser
3. Navigate between pages using the navigation menu

## Design System

- **Background**: Dark gradient (#050505 → #0b0b0b)
- **Primary Accent**: Red (#f11313)
- **Accent Hover**: Light red (#f53c3c)
- **Glass Effect**: rgba(255, 255, 255, 0.08)
- **Typography**: Arial, sans-serif

- Chrome (latest)

- Safari (latest)
